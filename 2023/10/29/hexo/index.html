GitHub Pges 建立 user-name.github.io 的 repo

如果不介意網域名稱，其實在這個此章節的內容，就可以成功建立個人化網站了。

  • 把做好的網頁秀在網路上，作為展示用途

組織 GitHub Repo

因為只有 GitHub 使用者名稱的專案，會自動形成網頁，所以其中的檔案會越來越繁雜。為了方便管理，建議每個專案分別建立，接著讓 GitHub Pages 的資料庫自動同步其他專案。

使用文章管理輔助工具，用 markdown 更新文章

可以使用 markdown 語法快速撰寫文章，節省從頭製作每個頁面的時間。
管理的工具有非常多種，而我最後選擇的是 Hexo，因為他是我找到唯一技術文件有簡體中文版本的文章管理輔助工具。
